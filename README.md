# PorousMediaFlow
Development of a case study regarding two-phase incompressible flow in porous media modeled by the Navier-Stokes equation for immiscible fluids.

References: Raeini, A.Q., Blunt, M.J., Bijeljic, B. (2012). Modelling two-phase flow in porous media at the pore scale using the volume-of-fluid method. Journal of Computational Physics. DOI: x.doi.org/10.1016/j.jcp.2012.04.011
